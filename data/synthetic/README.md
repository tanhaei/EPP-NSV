# Synthetic fixtures only

No patient-level data, BioArc data, clinical note corpus, or controlled-study result bundle is included in this repository. The executable generator creates privacy-safe synthetic patient--eye--episode pairs at runtime:

```bash
python -m epp_nsv.experiments --n-pairs 300 --seed 7 --out-dir outputs/experiment
```

The fixture labels are generated by the same versioned synthetic policy that the verifier evaluates. Therefore, their precision and recall values demonstrate implementation behavior only; they do not measure clinical accuracy, clinician agreement, or performance on a real EHR.

Do not commit de-identified or restricted records to a public Git repository unless the relevant approval, data-use agreement, governance process, and privacy review explicitly permit it. In most controlled clinical studies, this public repository should remain data-free.
