# No governed data belongs in this directory

This directory is an intentional guardrail. It contains no real BioArc data, no de-identified patient data, no free-text notes, and no adjudication material.

For a controlled study, store inputs only in the secure environment authorized by the relevant ethics approval and data-use agreement. Use the schemas in `docs/templates/` and the instructions in [`docs/CONTROLLED_STUDY.md`](../../docs/CONTROLLED_STUDY.md), but do not commit the resulting files here or anywhere in the public repository.
