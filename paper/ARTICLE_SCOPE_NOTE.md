# Article Scope Note

The public EPP-NSV repository supports two distinct publication tracks:

1. **Prototype / synthetic verification paper.** This is supported by the source code, unit tests, synthetic fixture generator, and public documentation.
2. **Governed clinical evaluation.** This is not completed or evidenced by the public repository. It requires private data access, a locked clinical policy, adjudicated pairs, correctly implemented baselines, statistical analysis, and archived run manifests.

Do not combine the abstract, tables, or conclusions of Track 2 with the artifacts of Track 1. In particular, real-data cohort sizes, clinical precision, clinician agreement, direct-LLM results, multimodal embedding results, and latency metrics must be generated from a documented controlled run, not inferred from synthetic tests.
