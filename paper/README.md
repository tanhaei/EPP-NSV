# Manuscript source and repository-scope note

`V1.tex` and `bibliography.bib` are the protocol-scope manuscript sources that motivated this public prototype. They are compatible with the repository only when the manuscript is framed as a methodological/software paper with synthetic verification.

The repository does **not** provide evidence for a completed BioArc cohort study, clinician-adjudication statistics, direct-LLM performance, multimodal embedding performance, or clinical-performance tables. Before a clinical-results manuscript cites this code, consult [`../docs/ARTICLE_ALIGNMENT.md`](../docs/ARTICLE_ALIGNMENT.md) and complete the controlled-study requirements in [`../docs/CONTROLLED_STUDY.md`](../docs/CONTROLLED_STUDY.md).

The `paper/` folder must not be used to silently switch the public repository's evidence boundary from synthetic software validation to real-data clinical validation.
